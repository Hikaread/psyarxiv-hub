## Summary
This paper introduces mIERda (multidimensional Insufficient Effort Responding detection algorithm), a machine learning approach using fair-cut random forests to detect insufficient effort responding (IER) in questionnaire data — a pervasive problem in clinical and research assessments that can inflate or distort measurement. Unlike existing methods that rely on arbitrary cut-off scores on single indices, mIERda combines multiple IER indices (IERIs) into a single continuous Insufficient Effort Responding Index (IERI) and uses a classification algorithm that outperforms sequential cut-off approaches.

Across four simulation studies (Study 1a-1c) and one empirical study (Study 2), the algorithm was tested under varying conditions: severity of IER (low, medium, high), prevalence (5-30%), sample sizes (500-5000), and six distinct IER response patterns (random, straight-lining, alternating, infrequent, floor, mixed). Key findings: mIERda achieved AUC values of 0.85-0.99 across simulation conditions, significantly outperforming the best sequential cut-off approach (balanced accuracy ~0.70-0.85). The algorithm was particularly robust at detecting mixed IER patterns (AUC = 0.97), which are the most common but hardest to detect. In the empirical validation using real questionnaire data, mIERda identified respondents with significantly lower standard deviations, shorter completion times, and more predictable response patterns compared to flagged-and-removed respondents.

The algorithm is implemented in an R package that takes raw item-level data as input and outputs a probability score for each respondent, requiring no pre-specified cut-offs from the user.

## Clinical Insight
IER is a critical threat to the validity of clinical assessments, symptom measures, and personality inventories used in therapy and research. A client who responds randomly or mechanically to a depression inventory, trauma questionnaire, or personality assessment can produce misleading results that may lead to incorrect case formulation, wrong treatment decisions, or erroneous research conclusions. mIERda offers a practical, cut-off-free solution that can be applied post-hoc to existing datasets. The key advantage over existing tools (such as long-string indices, Mahalanobis distance, or evenness indices used individually) is the combination of multiple IER indices into a single model, capturing heterogeneous response patterns that single-index approaches miss. For clinicians and researchers who use self-report measures, integrating mIERda screening into data processing pipelines could substantially improve data quality and diagnostic accuracy.

## Relevant For
Clinical researchers using self-report measures, psychometricians, assessment developers, program evaluators, any practitioner who relies on questionnaire data for clinical decision-making.

## Notes
R package available. Validated across 2,160 simulation conditions plus one empirical dataset. The algorithm performs best with sample sizes above 500. The authors note that IER detection should complement rather than replace other data quality procedures. Not yet tested on clinical diagnostic instruments specifically, though the underlying IER patterns are measure-agnostic.
